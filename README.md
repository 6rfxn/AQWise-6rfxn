# AQWise-6rfxn
Educational project for AQWise

## [Check Releases](https://github.com/6rfxn/AQWise-6rfxn/releases)

### Features
- Multi-language support (English, Indonesian, Malay)
- Settings page with drawer navigation
- Dark/Light theme toggle
- Bottom navigation bar
- User-friendly interface
- Tajweed Task with modern rule selection UI

## Built With
- **State Management:** Provider
- **Localization:** Flutter i18n
- **Storage:** SharedPreferences
